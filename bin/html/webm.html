#LiveOrStatic Of Android Screen (WebM Video)

  • You can click video to rotate locally. Double click to scale locally.
  • You can set querystring fps scale rotate to indicate framesPerSecond, size, rotation.

    Example: scale=0.5 scale=800x400 scale=800x scale=x400, rotate=90 rotate=270

  • You can set querystring fileIndex to indicate older file or specified file.

    Can be a number (bigger means older) or timestamp like 20140110_153928_704_000.

  • If the video is always BLACK, please turn on screen of your android device
  • This is just a sample HTML which contain a <video> tag to use ANOTHER URL for real video stream.

    So DO NOT use this page's URL as src of <source> of <video> directly. Please use:

    /@by?@q

If you can not see the video, please connect Android Device and Enabled Debug Mode of it(From Developer Setting), otherwise check log please.

If you can not see the video, maybe there are some server internal error such as file not found, disk full, check log please.
