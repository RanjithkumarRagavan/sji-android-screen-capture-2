    To Enable Authentication on Stream Web...
  1. Set adminKey in stream.json file to any string you like. e.g. "adminKey": "xxxx"
  2. Restart this application.
  3. Open this menu page by the URL appended with adminKey=yourAdminKey querystring. E.g. https://127.0.0.1/?adminKey=xxxx
  4. Use "Set Access Key" to attach access key to selected devices. (The Access Key is used as querystring accessKey=theKey of URL when access device's live capture or recorded files)
    To Enable Remote Access to Stream Web, ...
  1. Set ip = "*" or some explicit IP address in stream.json file.
  2. Optionally, set ssl.on = true to Enable SSL on Stream Web.
  3. Restart this application.
    To Enable SSL on Stream Web, ...
  1. Set ssl.on = true in stream.json file.
  2. Optionally, configure certificateFilePath or replace PKCS12 file ./ssl/StreamWeb.pfx with yours.
  3. Restart this application.
    To Enable Remote Access to Admin Web, ...
  1. Set adminWeb.ip = "*" or some explicit IP address in stream.json file.
  2. Optionally, set adminWeb.ssl.on = true to Enable SSL on Admin Web.
  3. Restart this application.
    To Enable SSL on Admin Web, ...
  1. Set adminWeb.ssl.on = true in stream.json file.
  2. Optionally, configure adminWeb.certificateFilePath or replace PKCS12 file ./ssl/AdminWeb.pfx with yours.
  3. Restart this application.
